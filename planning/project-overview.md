# The Human Neuron

## Vision
An interactive art experience where audience members each become an individual neuron in a live neural network. Through participation, attendees viscerally learn how AI works, how the brain processes information, and how neural networks function — by being part of one.

## Core Concept
- Each person acts as a single neuron in a collective neural network
- Participants use their phones (QR code scan) to join a live session
- The experience demonstrates:
  - How neurons activate within a network
  - How networks produce the "next token" in language models
  - How pathways form and strengthen through repeated use
  - How the network learns which pathways are good for what tasks
  - How those pathways get leveraged again (reinforcement / weight adjustment)

## Technical Approach
- QR code entry point → web-based session on participants' phones
- Cookie-based persistence so disconnected users can rejoin seamlessly
- Default accounts for each user (no sign-up friction)
- Real-time synchronization across all connected participants
- Visual representation of the network, activations, and signal flow
