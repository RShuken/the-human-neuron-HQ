# Research: Comparable Experiences & Prior Art

## Key Finding
No one appears to have done exactly this — making each audience member a literal neuron in a live neural network via their phones. This is genuinely novel.

## Closest Comparable Experiences

### 1. Crowd Participation Platforms (Kahoot-style)
- **Kahoot, AhaSlides, Mentimeter, Crowdpurr** — real-time phone-based audience participation
- These prove the QR-code-to-phone interaction model works at scale
- AhaSlides: real-time word clouds, polls, visual feedback as audience responds
- Crowdpurr: specifically designed for mobile-driven interactive live event experiences
- **Takeaway:** The input model is proven. What's new is making each participant a functional unit in a neural network.

### 2. Google's Teachable Machine
- Browser-based ML training — no coding required
- Uses MobileNet (pre-trained CNN) under the hood
- Great for educational workshops demonstrating ML concepts
- **Takeaway:** Could be a reference point for how to teach ML concepts visually. The Human Neuron goes further by making attendees *experience* it rather than just watch it.

### 3. Interactive Art Installations (2025-2026 Trends)
- **"Synthetic Memories"** by Steve Zafeiriou (MOMus Museum, Thessaloniki) — Inspire Project 2025
- **LACMA Art + Technology Lab** — launching new collaborative program structure in 2026
- Growing trend: AI-driven interactive installations where audience input shapes the artwork in real-time
- Motion tracking, touch-sensitive surfaces, AR, and AI enable real-time responsiveness
- **Takeaway:** The art world is moving toward participatory AI experiences. The Human Neuron fits squarely in this trend but is more educational/experiential.

### 4. Human-in-the-Loop Simulations
- Academic research on crowd sensing via mobile devices
- Real-time simulation augmented by human participation
- Agent-based frameworks combining simulation with real-world crowd sensing
- **Takeaway:** The technical underpinning exists in research. We're applying it as art/education.

## What Makes The Human Neuron Different
1. **Embodied learning** — you don't watch a neural net, you ARE one
2. **No app download** — QR code to web experience (low friction)
3. **Educational + experiential** — teaches real AI concepts through participation
4. **Collective emergent behavior** — the network only works because the audience works together
5. **Conference-ready format** — designed for a session, not a permanent installation

## Technical Precedents to Build On
- **WebSockets** for real-time bidirectional communication (proven at scale)
- **Cookie-based session persistence** for reconnection
- **Progressive web app patterns** for phone-based experiences
- **D3.js / Three.js / WebGL** for network visualization
- **Server-Sent Events or WebRTC** as alternatives for real-time sync
